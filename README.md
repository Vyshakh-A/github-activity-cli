# GitHub Activity CLI

A command-line tool that fetches and displays recent public GitHub activity for a user.

## Features
- Fetches public GitHub events
- Supports Push, Issues, Pull Requests, Forks, Watch, and Create events
- Clean, readable terminal output
- Linux-friendly (supports piping with grep)
- No external libraries

## Usage
```bash
github-activity <username>
github-activity <username> --limit 5
github-activity --help
